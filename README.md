#  Retail Sales Analytics & Decision Engine

##  About This Project

While learning Python and data analytics, I didn’t want to just build another basic project.
I wanted to create something that actually feels like a **real business solution**.

That’s how this project was born 

 A system that not only analyzes sales data but also **helps businesses make decisions**.

---

##  What This Project Does

This project takes raw retail sales data and turns it into meaningful insights:

✔ Calculates total revenue and key business KPIs
✔ Identifies top-performing and low-performing products
✔ Analyzes sales across locations and customer segments
✔ Provides smart discount strategies based on performance
✔ Recommends which products need restocking

##  Tech Stack

* Python
* NumPy

##  How It Works

1. Load sales data from a CSV file
2. Process and analyze using NumPy
3. Generate insights (KPIs, trends, performance)
4. Apply business rules to suggest decisions
